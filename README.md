# 📶 WiFi Issue Reporting System

A web-based system for reporting and tracking WiFi issues within a university.

## 📌 Description
This system allows users to report WiFi issues, track status, and enables administrators to manage and resolve problems efficiently.

## 🚀 Features
- Report WiFi issues
- Track issue status
- Admin management system

## 🖥️ System Workflow
User report → System store → Admin manage → Status update
