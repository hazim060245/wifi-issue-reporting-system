# WiFi Issue Reporting System

## Description
A web-based system for reporting and tracking WiFi issues within Sripatum University.

## Features
- Report WiFi issues
- Track issue status
- Admin panel for managing reports

## Tech Stack
- HTML, CSS, JavaScript
- SQL Database

## Purpose
This project was developed to improve the efficiency of reporting and resolving WiFi issues within the university.
